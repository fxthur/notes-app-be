## About 

Notes App Backend dibuat menggunakan NodeJS dan HAPI


### Requirement

* npm
  ```sh
  npm install npm@latest -g
  ```
### Installation

1. Clone the repo
   ```sh
   git clone https://github.com/fxthur/notes-app-be
   ```
2. Install NPM packages
   ```sh
   npm install
   ```
3. Run 
   ```js
   npm start dev
   ```


## Feature
- Add Note
- Change Note
- Delete Note
- Show Note
- Filter by Id Note
